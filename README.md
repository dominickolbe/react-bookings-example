## Getting Started

### Prerequisites

I build this entire project with the following setup:

- macOS Big Sur version 11.1
- node v12.20.0
- npm v6.14.8
- yarn v1.22.10

### Development

```bash
yarn install
```

## Run project

4. start frontend application

```bash
yarn start
```

## Testing

tests are more or less useless, but I wanted to show how I'd write tests.

```
yarn test
```
